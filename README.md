# Mineração de Dados

Este repositório contém meus projetos desenvolvidos durante a disciplina de Mineração de Dados

## Estrutura
- `DataMining_02_understanding_and_preprocessing.ipynb` -- Entendendo os dados e realizando pré-processamentos básicos.
- `DataMining_03_association_rules.ipynb` -- Trabalho de regras de associação de produtos de um supermercado.

## Observação
Os datasets utilizados não estão incluídos neste repositório.

## Tecnologias
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn
- NLTK / spaCy
- Jupyter Notebook
