# Mineração de Dados

Este repositório contém meus projetos desenvolvidos durante a disciplina de Mineração de Dados

## Estrutura
- `DataMining_02_understanding_and_preprocessing.ipynb` -- Entendendo os dados e realizando pré-processamentos básicos.
- `DataMining_03_association_rules.ipynb` -- Trabalho de regras de associação de produtos de um supermercado.
- `DataMining_04_clustering.ipynb` -- Agrupamento de jogos através de atributos de design.
- `DataMining_05_classification.ipynb`-- Classificação binária de perfomance na mesma base de dados que (DataMining_2).
- `DataMining_07_solving.ipynb` -- Trabalho final da disciplina: problema de classificação em bases de dados comuns com análise de top-K com GridSearch e Otimização Multiobjetivo.

## Observação
Os datasets utilizados não estão incluídos neste repositório.

## Tecnologias
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn
- NLTK / spaCy
- Jupyter Notebook
